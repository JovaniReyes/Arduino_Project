Overview

The quadruped robot was designed with 3D-printed parts and custom components. It is controlled using an Arduino and features obstacle avoidance capabilities.

Design Process

The robot's parts were printed using PLA plastic. Servo motors are used for movement, controlled by a program written in C for Arduino.

Circuit Details

The custom-made power supply includes voltage regulation, ensuring stable power for all the components.
