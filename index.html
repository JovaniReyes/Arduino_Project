Quadruped Arduino Robot

Gallery Parts Documentation

Project Overview

This quadruped robot was built from scratch using 3D printed parts, an Arduino board, and custom soldered circuitry. The robot is designed to move using 12 servo motors, and it is controlled via Bluetooth. It uses an ultrasonic sensor to avoid obstacles.

Parts & Program

The robot consists of several key components including a micro Arduino board, servo motors, a Bluetooth module, and a custom-made power supply circuit. Check the 'Parts' section for more details.
